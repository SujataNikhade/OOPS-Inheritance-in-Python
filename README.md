# Python OOP Inheritance Practice

This repository contains well-structured Python programs demonstrating different types of Object-Oriented Programming (OOP) inheritance.
It is designed for students, beginners, and anyone wanting to understand inheritance in Python clearly.

##  Concepts Covered

### Single Inheritance

Child class inherits from one parent class.

### Multilevel Inheritance

Grandparent → Parent → Child hierarchy.

### Multiple Inheritance

A class inherits from two or more parent classes.

### Hierarchical Inheritance

One parent class with multiple child classes.

### Hybrid Inheritance

Combination of multiple and multilevel inheritance.
